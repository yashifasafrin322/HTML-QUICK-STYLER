HTML Quick_Styler
 Dynamic HTML Page Generator 🎨

HTML Quick Styler is a Python-based web application that dynamically generates styled HTML pages using predefined modern color palettes. 

The system allows users to input custom content such as title and description, select a theme, and instantly generate a responsive and visually appealing HTML layout.

This project demonstrates how backend logic written in Python can be integrated with frontend technologies to dynamically render styled web pages through an interactive interface.

 Project Overview

The application uses structured color palette dictionaries and a dynamic HTML generation function to construct clean and responsive webpage layouts.

It is built using Gradio, enabling users to interact with the system through a simple web interface without writing CSS manually.

The tool simplifies the HTML styling process and improves productivity for beginners learning frontend development concepts.

Core Features

- Dynamic HTML page generation
- Predefined modern UI color palettes
- Real-time preview interface
- Responsive layout structure
- Beginner-friendly design
- Clean and minimal UI

🛠️ Technologies Used

- Python 3.8+
- Gradio – Web interface framework
- HTML5 – Structure
- CSS3 – Styling
- Google Colab – Development environment
- GitHub – Version control and hosting

⚙️ System Workflow

1. User enters title and description.
2. User selects a color palette.
3. Backend Python function generates styled HTML dynamically.
4. Output is rendered instantly in the Gradio interface.

🎯 Project Objective

The main objective of this project is to:

- Demonstrate dynamic web page generation using Python.
- Simplify frontend styling for beginners.
- Integrate backend logic with UI frameworks.
- Build a practical mini web development tool.
